假設a=212  
位數n=3  
k=10^(n-1)=100  

#### 計算** 0~9 的個數  
** = 20 個 0 ~ 9 ( 十位數字10個，個位數字重複10次 )  
*** = 300 個 0 ~ 9 (百位數字重複100次，後面加上10組** = 100+10*20)   
**** = 4000 個 0 ~ 9  

以此類推  -> 0 ~ 9 的個數 = (k / 10) x ( *的個數 )  

#### 首位出現的個數
0\*\* -> 0 出現 100 (k) 次 (\*\* = 0\~99)  
1\*\* -> 1 出現 100 (k) 次  
2\*\* -> 2 出現 13 次 = 12 (0\~11) + 1 (12)   

#### 首位固定，後面出現的個數
0\*\* -> 0 \~ 9 出現 20 次  
1\*\* -> 0 \~ 9 出現 20 次  

#### 第二位出現的個數 k=10   

20\* -> 0 出現 10 (k) 次 (\* = 0\~9)  
21\* -> 1 出現 3 次 = 2 (0~1) + 1 (2)  

#### 第二位固定，後面出現的個數
20\* -> 0 ~ 9 出現 1 次  

#### 第三位出現的個數 k=1   
210 -> 0 出現 1 (k) 次
